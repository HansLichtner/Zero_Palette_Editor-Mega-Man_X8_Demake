# Zero Palette Editor for Mega Man X8 Demake

## A simple palette editor for Zero custom in Mega Man X8 Demake

### Instructions:
- Place the executable file in the same folder as Mega Man X8 Demake

### Functions:
- Apply palette (save current palette as "custom_zero_palette.txt")
- Save palette (save current palette as "custom_zero_palette_<palette_name>.txt")
- Load palette (open a custom palette .txt file)
- Reload palette (reload the currently open palette)
- Reset palette (reset to default palette)
